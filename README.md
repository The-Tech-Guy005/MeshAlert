# MeshAlert

MeshAlert is an Android application that enables offline communication during disasters by creating a mesh network between nearby smartphones using Bluetooth and Wi-Fi Direct.

## Problem

During disasters such as earthquakes, floods, and cyclones, mobile towers and internet services often become unavailable, making it difficult for people to communicate or request help.

## Solution

MeshAlert allows nearby Android devices to communicate directly without relying on the internet or cellular networks. Emergency messages can be relayed from one device to another until they reach rescue teams or other users.

## Features (Current)

- Home Screen
- SOS Screen
- Safe Screen
- Broadcast Screen
- Navigation between screens
- Back navigation
